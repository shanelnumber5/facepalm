About This Project

This is a TTRPG character collection for the Character Creation Challenge (May 2026). The format is a facepalm-style group page which is designed to feel familiar and recognizable.

These are not real people. If you think you recognize someone, that says more about the environment than the individual. These archetypes are built from repeated online behaviors: belief spirals, soft pitches, nostalgia loops, and vague posts that generate more theories than answers.

The TTRPG stats are mapped to social media behaviors on purpose. Wisdom fluctuates. Charisma carries more than it should. Skills are exaggerated, but not invented. This is a system applied to something that already behaves like one.

There are no hit points. Nothing here resolves cleanly enough for that.

All artwork is hand-drawn. Profile images are embedded directly into the page.

# facepalm

**A TTRPG character collection disguised as a small-town social feed.**

I do not have the app. I use the browser version out of principle, which is what people say when they are definitely still checking the drama.

**facepalm** is a browser-based character gallery made for the TTRPG Character Creation Challenge. Each profile is a fictional archetype built from repeated online behaviors: public vagueposting, civic overexplaining, soft pitches, conspiracy spirals, nostalgia loops, and the kind of comment-section escalation that makes you close the tab and immediately reopen it.

These are not real people.

If you think you recognize someone, that says more about the environment than the individual.

## How to Use

Click through the profiles like you are browsing a local social feed you absolutely did not mean to spend twenty minutes reading.

Each character includes:
- a banner image
- an archetype label
- an About section
- TTRPG-style stats
- skills
- a special ability
- a representative post

There are no hit points. Nothing here resolves cleanly enough for that.

## Content Note

This project satirizes online behavior, public posting habits, small-town social dynamics, and social media archetypes. It does not depict real individuals.
